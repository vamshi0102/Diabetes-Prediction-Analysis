# Diabetes Prediction Analysis

## Project Overview

This project focuses on predicting diabetes using the Pima Indians Diabetes Dataset. The objective is to analyze important health-related factors and build a machine learning model that can predict whether a patient is likely to have diabetes.

## Dataset

* Dataset: Pima Indians Diabetes Dataset
* Total Records: 768
* Features: 8 Input Features + 1 Target Variable

## Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Google Colab

## Exploratory Data Analysis (EDA)

The following analyses were performed:

* Diabetes Distribution
* Glucose Level vs Diabetes
* BMI vs Diabetes
* Age vs Diabetes

## Key Insights

* Higher glucose levels are strongly associated with diabetes.
* Patients with higher BMI values are more likely to have diabetes.
* Older individuals show a higher risk of diabetes.
* Approximately 35% of patients in the dataset were diabetic.

## Machine Learning Model

**Model Used:** Logistic Regression

## Model Performance

**Accuracy:** 75.97%

### Classification Results

* Non-Diabetic Prediction Accuracy: Strong Performance
* Diabetic Prediction Accuracy: Moderate Performance
* Overall Model Accuracy: 75.97%

## Conclusion

The analysis identified glucose level, BMI, and age as major factors influencing diabetes risk. A Logistic Regression model was developed to predict diabetes outcomes with an accuracy of 75.97%. The project demonstrates how data analytics and machine learning can support early diabetes detection and healthcare decision-making.
